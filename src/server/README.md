# Backend - Lógica del Servidor

Esta carpeta contiene toda la lógica del backend del proyecto Next.js.

## Estructura

- **controllers/**: Controladores que manejan la lógica de las rutas API
- **services/**: Servicios de negocio (acceso a BD, lógica compleja)
- **middleware/**: Middlewares personalizados (autenticación, validación, etc)
- **utils/**: Funciones utilitarias del servidor
